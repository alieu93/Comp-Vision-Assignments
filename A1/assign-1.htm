Q1: Playing with Frequencies [70 marks]

Develop an application for performing focus analysis on an image. Focus or lack thereof is an important tool employed by photographers to create aesthetically pleasing photographs. While we are far from discovering automated schemes for evaluating aesthetic scores of an image, we can definitely identify regions in an image that are in sharp focus. Your goal is to identify regions in an image that are in sharp focus. You can draw a rectangle on the image to identify regions of sharp focus as seen below.

Your code should load a file and show regions are in sharp focus. You should use Python with OpenCV for this purpose. So the program will work as follows:

$ python focus.py test-image.jpg
Focus analysis took 200 ms.
press any key to quit

The program will load and display the image with highlighted regions and wait for users input. The program will also print a message Focus analysis took 200 ms. that will tell us how long it took to perform focus analysis. The program will quit when a key is pressed.
